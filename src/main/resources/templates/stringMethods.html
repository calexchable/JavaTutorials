With one parameter, the slice() method starts at the noted position until the end of the string. If parameter is negative it counts form the end of the string (but still returns from that positon to the end of the string.

Substring Method

The substring() method extracts a part of a string and returns the extracted parts in a new string. Similar to slice, it, however, cannot accept negative indexes.

substr() method

Similar to slice, but second parameter is length of extracted part.

If you delete the second parameter, it returns until the end of the string.

Negative parameter the position counts from the end of the string.
